An OSGeo Community project

GeoStyler is an Open Source JavaScript library that enables users to style maps with the help of a Graphical User Interface. The goal of GeoStyler is a universally applicable GUI that works with any given style format; may it be SLD, QML, OpenLayers, etc.

The demo https://geostyler.github.io/geostyler-demo/

Furthermore, GeoStyler allows for the translation between multiple styling formats, i.e. SLD, OpenLayers, QGIS, Mapbox.

This is a very important “furthermore”! GeoStyler allows the conversion, and therefore reuse of style between different applications.

See the Supported Formats page for links to all the parsers.

Getting familiar with Discord, Visual Studio Code, TypeScript, ES^, `Jest <>`_ and Jest Runner (thanks to @jansule for helping me set this up). There’s nothing like a codesprint for learning new tools, workflows, and languages.

GeoCat library is written in Python.

The GeoStyler project is written in TypeScript and transpiled to JavaScript. This makes all the functionality available in the browser.
